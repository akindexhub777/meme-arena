# ⚔️ Meme Arena

**La première arène de memes jugée par Intelligent Contracts sur GenLayer.**

Soumets ton meme → L’IA de GenLayer l’analyse et lui donne un score objectif → Affronte la communauté en temps réel.

---

## ✨ Fonctionnalités

- **Submission de memes** avec image + description
- **Scoring automatique par IA** grâce à GenLayer Intelligent Contract
- **Classement en temps réel** (Top Memes)
- **Design moderne dark** avec accents purple/cyan
- **Prêt pour le testnet GenLayer** (Asimov / Bradbury)
- Système de votes
- Architecture facilement extensible (récompenses, staking, battles 1v1…)

---

## 🛠️ Tech Stack

### Frontend
- HTML5 + Tailwind CSS
- Vanilla JavaScript
- Responsive & mobile-first
- Design dark mode compétitif

### Backend (GenLayer)
- **Intelligent Contract** écrit en Python
- Utilisation de `gl.llm.generate_structured()` pour le jugement IA
- `TreeMap` pour le stockage des memes
- Consensus via les validators LLM de GenLayer
